### 2. 資料前處理工具設計（`utils/`）

#### 工作目標
- 建立可重複使用的前處理工具，支援行人偵測與屬性分析任務。
- 確保資料格式一致、標註解析正確，並具備基本資料增強能力。
- 提供單元測試，確保工具穩定可靠。

1. **資料集格式分析與規劃**

   - 設計統一的資料結構（如 class 或 dict），方便後續處理。

    - 盤點資料來源（公開資料集、自建資料集）。
    - 分析標註檔案格式（COCO、VOC、CSV、JSON 等）。
    - 設計統一的資料結構（如 Python class 或 dict），方便後續處理。
